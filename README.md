# Habit Tracker

A full-stack habit tracking application with social accountability.

## Tech Stack
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express + Prisma
- Database: PostgreSQL

## Setup Instructions

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Update DATABASE_URL in .env
npm run migrate
npm run dev
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```
